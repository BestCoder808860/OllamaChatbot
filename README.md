# 🧠 Ollama Chatbot (Windows EXE)

A simple **offline AI chatbot** powered by Ollama + Llama models.

This project contains only:
- `OllamaChatbot.zip` (a ZIP file with the app)
- `README.md` (this guide)

No coding required to use it.

---

# 📌 What you need BEFORE running

This chatbot will NOT work unless you install Ollama first.

## 1. Install Python (ONLY if you want to rebuild it)
👉 https://www.python.org/downloads/

(Not needed just to run the EXE)

---

## 2. Install Ollama (REQUIRED)

Download Ollama here:
👉 https://ollama.com/download

Install it normally like any Windows app.

---

## 3. Download a Llama model

Open Command Prompt and run:

```bash
ollama pull llama3.2
